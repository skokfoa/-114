# Program R 最小路徑

此專案是一個使用 Python 撰寫的迷宮最短路徑求解器，目標是計算從迷宮起點到終點所需的最小移動成本。

## 功能介紹
- 使用 **Dijkstra 演算法** 進行路徑搜尋，確保找到路徑權重總和最小的走法。
- 支援動態輸入迷宮尺寸與權重陣列。

## 如何使用
1. 執行程式：`python Program_R.py`
2. 輸入迷宮維度 `n` (行) 和 `m` (列)。
3. 依序輸入迷宮的權重數值。

## 演算法說明
- 程式透過 `heapq` 維護一個優先佇列 (Priority Queue)，並使用 `min_cost` 二維陣列記錄目前到達各點的最小成本，以達到 $O(E \log V)$ 的效率。

# Program S最小生成樹

此專案旨在解決「最小生成樹 (Minimum Spanning Tree, MST)」問題，透過優化道路連接方式來計算可以省下的電費。

## 功能介紹
- 實作 **Kruskal 演算法**：使用並查集 (Union-Find) 判斷邊是否會形成環，適合稀疏圖。
- 實作 **Prim 演算法**：使用優先權佇列 (Priority Queue) 逐步擴展生成樹，適合稠密圖。
- 支援多筆測資 (Test Cases) 處理。

## 使用方式
1. 執行程式：`python Program_S.py`
2. 輸入測試資料筆數 `T`。
3. 依照提示輸入路口數量 `m` 與道路數量 `n`。
4. 輸入所有道路的連接路口 `x`, `y` 以及權重 `z`。
5. 根據提示選擇要使用的演算法 (1: Kruskal, 2: Prim)。

## 演算法邏輯
- **Kruskal**：先對邊進行排序，利用 `Union-Find` 檢查連通性。
- **Prim**：從單點出發，每次挑選連接已訪問區域的最小邊。
